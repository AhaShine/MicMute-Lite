# MicMute Lite

Small Windows tray app for quickly muting and unmuting your microphone.

## Features

- toggle, push-to-talk, and push-to-mute modes
- keyboard and mouse hotkeys, including side mouse buttons
- tray control
- small settings window
- movable overlay
- custom on/off sounds: put `on.mp3` and `off.mp3` next to the exe
- autostart option

## Download

Download `MicMuteLite.exe` from [Releases](https://github.com/AhaShine/MicMute-Lite/releases).

## Build

```bat
run.bat
build.bat
```
